# AWS S3 File Upload & Download System

A full-stack application that allows users to upload and download files using AWS S3 storage.

Built with:
- React (Frontend)
- Node.js + Express (Backend)
- AWS S3 (Cloud Storage)

---

## 🚀 Features

- Upload files to AWS S3
- Download files securely
- REST API integration
- Full-stack cloud application

---

## 🛠 Tech Stack

- React
- Node.js
- Express.js
- AWS SDK
- Axios

---

## 📋 Prerequisites

- Node.js installed
- AWS account
- S3 bucket configured
